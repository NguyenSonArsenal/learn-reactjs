## SYSTEM REQUIREMENT

* Node
  - 14.17.3
* NPM 
    - 6.14.13
* Yarn
  - 1.12.17
  
## INSTALL
### `npm install`

## RUN
### `npx json-server --watch db.json --port 3002` on 1 tab terminal
### `npm start` on 1 tab terminal
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
